    網站抵制小幫手 Safari Extension，幫你阻擋你想抵制的網站

    Project maintained by zhusee2 Hosted on GitHub Pages — Theme by mattgraham

    《網站抵制小幫手》讓您可以自由設定要抵制的網站，往後再度瀏覽到設定抵制的網站時，小幫手就會提醒您繼續抵制。《網站抵制小幫手》目前提供 Safari Extension 版本。

    網址過濾格式

    透過《網站抵制小幫手》工具列按鈕新增抵制網站時，預設會抵制整個 domain。但是您也可以指定為以下格式：

    1. *.bad.domain.com
    2. bad-website.xxx/horrible/*
    3. example.com

    如何取消抵制特定網站

    在《網站抵制小幫手》工具列按鈕的選單中，選擇「管理抵制清單⋯」，即可進入編輯抵制清單。抵制清單目前僅提供純文字編輯介面，採取 JSON 格式。請以字串方式存放如上過濾格式，並將字串組成一陣列。範例如下：

    [
  "*.bad.domain.com",
  "bad-website.xxx/horrible/*",
  "example.com"
]
